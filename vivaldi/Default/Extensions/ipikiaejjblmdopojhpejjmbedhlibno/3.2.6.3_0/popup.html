There are four ways to use Spreed on text

Extension Icon

  • 1. Open the Spreed menu by clicking on the Spreed extension icon.
  • 2. Select "Spreed current website".

Auto-extract, Alt+V (default hotkey)

  • Just hit Alt+V to automatically extract the main content of a webpage and open Spreed!

Select Text

  • 1. Select text on the webpage with your mouse.
  • 2. Right click, then select "Spreed selected text" in the menu.
  • OR hit Alt+Vto open Spreed with the selected text.

Copy-paste Text

  • 1. Copy text from anywhere (e.g. PDFs).
  • 2. Open the Spreed menu by clicking on the Spreed extension icon in the browser.
  • 3. Select "Paste text into Spreed".
